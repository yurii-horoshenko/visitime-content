# Архитектура контент-пайплайна

## Общая схема

```
┌─────────────────────────────────────────────────────────┐
│                   КОНТЕНТ-ПАЙПЛАЙН                       │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  1. СТРАТЕГИЯ           2. СОЗДАНИЕ         3. ПУБЛИКАЦИЯ│
│  ┌──────────────┐      ┌──────────────┐    ┌───────────┐│
│  │ Narratologist │──►   │ Content      │──► │ Platform  ││
│  │ (арка серии)  │      │ Creator      │    │ Optimizer ││
│  └──────┬───────┘      │ (контент-план)│    │ (адапт.)  ││
│         │              └──────┬───────┘    └─────┬─────┘│
│         ▼                     ▼                   ▼      │
│  ┌──────────────┐      ┌──────────────┐    ┌───────────┐│
│  │ Social Media  │      │ Short-Video  │    │ SEO       ││
│  │ Strategist   │      │ Editing Coach│    │ Specialist││
│  │ (расписание) │      │ (монтаж)     │    │ (теги)    ││
│  └──────────────┘      └──────────────┘    └───────────┘│
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## Компоненты

### 1. Стратегический слой

| Компонент | Агент | Ответственность |
|-----------|-------|----------------|
| Нарратив серии | Narratologist | Общая арка: от идеи VisiTime до запуска |
| Контент-план | Content Creator | Темы эпизодов, хуки, CTA |
| Кросс-платформенная стратегия | Social Media Strategist | Расписание, частота, best time to post |

### 2. Продакшн слой

| Компонент | Агент | Ответственность |
|-----------|-------|----------------|
| Монтаж гайдлайны | Short-Video Editing Coach | Рекомендации по монтажу каждого видео |
| Визуальный сторителлинг | Visual Storyteller | Нарративная структура внутри видео |

### 3. Дистрибуция слой

| Компонент | Агент | Ответственность |
|-----------|-------|----------------|
| TikTok оптимизация | TikTok Strategist | Тренды, звуки, хештеги TikTok |
| Instagram оптимизация | Instagram Curator | Reels-специфика, сообщество |
| YouTube оптимизация | Video Optimization Specialist | Shorts SEO, тамбнейлы |
| Общий SEO | SEO Specialist | Кросс-платформенные хештеги, описания |

## Workflow (цепочка)

```
PM (задача) → Narratologist (арка) → Content Creator (план видео)
  → Short-Video Editing Coach (монтаж) → Visual Storyteller (визуал)
  → TikTok/Instagram/YouTube Strategists (адаптация)
  → SEO Specialist (теги) → Social Media Strategist (расписание)
  → QA Engineer (финальная проверка)
```
