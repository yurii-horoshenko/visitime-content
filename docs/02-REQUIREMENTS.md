# Требования

## Функциональные требования

### FR-1: Планирование контента
- Генерация контент-плана на неделю/месяц
- Определение тем эпизодов на основе прогресса VisiTime
- Привязка к актуальным трендам платформ

### FR-2: Создание контента
- Генерация сценариев для каждого видео (хук, основа, CTA)
- Рекомендации по монтажу (темп, переходы, текст на экране)
- Адаптация контента под каждую платформу

### FR-3: Оптимизация под платформы
- TikTok: тренды, звуки, хештеги, время публикации
- Instagram Reels: визуальный стиль, хештеги, сообщество
- YouTube Shorts: SEO, описания, теги, тамбнейлы

### FR-4: Публикация
- Расписание публикаций (best time to post)
- Кросс-постинг стратегия (что адаптировать, что уникально)
- Чек-лист перед публикацией

## Нефункциональные требования

### NFR-1: Частота
- Минимум 3 видео в неделю на каждую платформу
- Контент-план готов минимум на неделю вперёд

### NFR-2: Качество
- Каждое видео проходит QA-чек (хук, длина, CTA, теги)
- Соответствие гайдлайнам платформ (размер, длительность, формат)

### NFR-3: Консистентность
- Единый визуальный стиль серии
- Узнаваемый формат (intro, outro, font, цвета)
- Связность нарратива между эпизодами

## Ограничения

- Один человек (автор = разработчик = монтажёр)
- Бюджет: $0 (органический рост)
- Время на контент: ~1ч/день максимум
- Язык: английский (основной) + русский (опционально)
