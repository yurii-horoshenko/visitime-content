# VisiTimeContent

## Описание

Контент-проект для продвижения **VisiTime** — платформы бронирования услуг (iOS + Android + KMM + .NET 10 backend). Серия коротких видео в формате "Build in Public", документирующая процесс разработки от идеи до запуска.

## Цели

1. **Awareness** — познакомить аудиторию с VisiTime до запуска
2. **Community** — собрать аудиторию разработчиков и потенциальных пользователей
3. **Content flywheel** — создать систему регулярного контента с минимальными затратами времени
4. **SEO/Discovery** — обеспечить присутствие во всех основных short-video платформах

## Целевая аудитория

- **Основная:** Инди-разработчики, стартаперы (build in public community)
- **Вторичная:** Потенциальные пользователи VisiTime (малый бизнес, фрилансеры)
- **Третичная:** Tech-энтузиасты, подписчики dev-контента

## Платформы

| Платформа | Формат | Частота | Фокус |
|-----------|--------|---------|-------|
| TikTok | 15-60s vertical | 3-5/неделю | Вирусный, трендовый контент |
| Instagram Reels | 15-90s vertical | 3-5/неделю | Визуальный сторителлинг |
| YouTube Shorts | 15-60s vertical | 3-5/неделю | SEO, долгий хвост |

## Формат серии

**"Building VisiTime"** — серия эпизодов, каждый раскрывает один аспект разработки:
- 🎯 Хук (0-3s) — цепляющее начало
- 📖 Контент (3-50s) — основная часть
- 🔥 CTA (последние 5s) — призыв к действию

## Стек (инструменты)

- Съёмка: смартфон / screen recording
- Монтаж: CapCut / DaVinci Resolve
- Планирование: AI-агенты (этот проект)
- Публикация: нативные платформы + кросс-постинг
