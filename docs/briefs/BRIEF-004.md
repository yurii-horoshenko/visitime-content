# BRIEF-004 — YouTube Short: "VisiTime за 60 секунд"

## Goal
Product tour — максимально быстрое и ясное знакомство с продуктом. Аудитория: люди, уже знакомые с проблемой (пришли из TikTok/Reels), ищут решение. Цель: понимание продукта + клик по ссылке / установка.

## Platform + Format
- Платформа: YouTube Shorts
- Формат: вертикальное видео 9:16, 55-60 секунд (YouTube Shorts max)
- Пиллар: App in Action

## Hook (первые 3 секунды)
Экранзапись: запуск VisiTime, чистый интерфейс.
Голос:
> "Это VisiTime. Покажу всё за 60 секунд."

Текст оверлей: "Честный обзор без воды."

## Script Outline
1. **Хук (0-3 сек)** — прямо заявить: "60 секунд, всё самое важное"
2. **Проблема в 1 предложении (3-8 сек)** — "Большинство людей не знают, куда уходит время. Это исправляет VisiTime."
3. **Фича 1 (8-20 сек)** — Трекинг задач: как начать таймер, как выглядит запись. Показ в реальном интерфейсе.
4. **Фича 2 (20-35 сек)** — Weekly report: что видно на дашборде, категории, инсайты
5. **Фича 3 (35-50 сек)** — Мобильность: работает на телефоне, синхронизация, минималистичный UX
6. **Итог + CTA (50-60 сек)** — "Бесплатно. Ссылка ниже. Попробуй 7 дней."

## Visual Notes
- 100% экранная запись реального приложения + голос за кадром
- Каждая фича — отдельный "блок" с заголовком (текст оверлей)
- Переходы: быстрые cut, без анимаций (ощущение темпа)
- Оверлей таймера (0:00 / 1:00) в углу — подчёркивает обещание "за 60 секунд"
- Без музыки или очень тихая фоновая — голос должен быть разборчив
- Закрывающий кадр: логотип VisiTime + ссылка на скачивание

## CTA
> "Ссылка в описании — попробуй 7 дней бесплатно."

Текст описания: "VisiTime — time tracker для фрилансеров и продуктивных людей. Установи и посмотри, куда уходит твоё время → [ссылка]" + хэштеги: #visitime #timetracker #продуктивность #appреview #фриланс

## Acceptance Criteria
1. Видео ровно 55-60 секунд (не обрезается YouTube Shorts)
2. Показаны минимум 3 ключевых фичи продукта в реальном интерфейсе
3. CTR по ссылке в описании > 3% от просмотров
4. Видео понятно без звука (субтитры или текст-оверлей покрывают все ключевые моменты)
