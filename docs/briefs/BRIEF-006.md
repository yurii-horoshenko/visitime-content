## Script Outline
1. **Введение (300 слов)** — боль фрилансера: время = деньги, но куда уходит время — загадка. Главный тезис: трекинг — первый шаг к управлению
2. **Почему фрилансерам нужен time tracker (500 слов)**
   - Финансовые потери от неучтённого времени (математика из BRIEF-003)
   - Биллинг клиентам: точность vs ощущение
   - Налоги и отчётность: документирование рабочего времени
3. **Как выбрать time tracker: 5 критериев (500 слов)**
   - Простота старта (нет времени на настройку)
   - Мобильность (работа не только за столом)
   - Отчёты и визуализация
   - Интеграции (Trello, Notion, Jira)
   - Цена vs функционал
4. **VisiTime: обзор для фрилансера (600 слов)**
   - Ключевые фичи с реальными скриншотами
   - Как выглядит типичная рабочая неделя в VisiTime
   - Что показывает первый отчёт (конкретные инсайты)
   - Плюсы и минусы (честно — доверие аудитории)
5. **Практика: 7-дневный план запуска трекинга (400 слов)**
   - День 1-2: установка, настройка категорий
   - День 3-5: просто трекай без оценки
   - День 6-7: первый анализ → один вывод → одно изменение
6. **FAQ (300 слов)** — 5 частых вопросов с ответами (SEO-фрагменты для featured snippets)
7. **Заключение + CTA (200 слов)** — "Не нужна идеальная система. Нужна видимость. Начни с 7 дней." + кнопка установки
