### Понедельник 04.05 · пост #7 — «🏠 Платформа для solo-мастеров»

- **Тип:** длинный текст + скринкаст [EN] + IG-карусель [RU]
- **Платформы:** LinkedIn + Instagram + Facebook
- **О чём:** расширяем позиционирование. «VisiTime не только для больших салонов. На платформе регистрируются solo-мастера: девочки с ресницами на дому, частные парикмахеры, мастера маникюра, массажисты, репетиторы.»
- **Посыл для solo-мастеров:**
  - Больше не надо вести переписку в DM — у тебя есть витрина
  - Автоматические напоминания клиентам — меньше no-show
  - Онлайн-оплата — деньги до визита
  - Клиенты находят через фильтры — не только через знакомых и Instagram
- **Визуал:** до/после — «как было: DM-хаос, показ примеров в Stories» → «как стало: профиль с ценами, расписанием, оплатой»

- **Агент:** `content-creator` + `visual-storyteller`
