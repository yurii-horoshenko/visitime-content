# План съёмок — Недели 1–2

**Контекст:** стратегия переработана — 10 из 14 постов не требуют съёмки (текст + carousel + screencast). Остаётся **1 съёмочный день** на 4 видео + b-roll на 2 недели.

**Языки видео:** RU (Instagram/TikTok) + EN субтитры для LinkedIn/YouTube. Польского нет.

Принцип — **batch shooting**: один раз настроили свет/фон/реквизит — сняли пачку.

## Общая инфраструктура продакшн

### Файл-менеджмент
```
production/
├── raw-footage/
│   ├── day-1-founder/
│   │   ├── a001_clip001.mov     # по naming convention
│   │   └── ...
│   └── day-2-salon/
├── edited/
│   └── 01-dlaczego-buduje/
│       ├── 01-master.mp4        # 9:16 TikTok/IG
│       ├── 01-square.mp4        # 1:1 FB
│       └── 01-thumbnail.png
└── audio/
    ├── music-library/
    └── voice-overs/
```

### Bitrate / формат
- Export: H.264, 25 Mbps, 1080×1920 (9:16) / 1080×1080 (1:1) / 1920×1080 (16:9)
- Sound: -14 LUFS (IG/TikTok loudness)
- Captions: burned-in RU для IG/TikTok · EN versions для LinkedIn/YouTube (нет PL)

### Утилиты
- **CapCut** — быстрый монтаж, AI captions
- **DaVinci Resolve** — финишинг, color
- **Descript** — voice cleanup, filler word removal
- **Epidemic Sound** — music ($15/мес)
- **Canva / Figma** — thumbnails, carousels

### Бюджет продакшн месяц 1-2
| Позиция | Сумма |
|---------|-------|
| Музыкальная подписка | $15 × 2 = $30 |
| Actress / модель (опционально) | 400 PLN × 1 |
| Реквизит / fabric / props | ~200 PLN |
| Microphone (если нет) | ~500 PLN one-time |
| **Итого** | ~1000-1500 PLN / ~€250-350 |
