## ⚠️ Важно о языках

**Контент делаем на 2 языках: RU + EN. Польского не делаем** — основатель по-польски не говорит, аутентичного контента на PL на старте не получится. Польский появится когда:
- наймём Polish content-creator, или
- появятся первые клиенты-партнёры (user-generated на PL)

| Язык | Платформы | Аудитория |

## 🎯 Позиционирование VisiTime

**Экосистема из двух приложений для бронирования любых услуг:**

- **🧡 VisiTime (Client App)** — поиск, фильтрация, бронирование, оплата, отзывы
- **💼 VisiTime Business** — для провайдеров: витрина, команда, CRM, аналитика, подписки Free→Enterprise

### 5 столпов позиционирования

1. **🌐 Фильтр по языку мастера** (UA/RU/EN/PL/KO) — killer для диаспоры
2. **👤 Фильтр по гендеру мастера** — комфорт без объяснений
3. **📅 Все записи дня в одном месте** — автосервис, врач, парикмахер, садик — видно вместе
4. **🏠 От салона до solo-мастера** — крупные сети, малые студии, девочки с ресницами дома
5. **🌍 100+ типов услуг** — не только beauty: auto, childcare, медицина, образование, fitness — через Google Places

## Две аудитории — два потока

### 💼 B2B — провайдеры услуг (EN primary)

- **Кто:** от solo-мастеров на дому до средних салонов/студий
- **Платформы:** LinkedIn (primary), Twitter/X, YouTube (tutorials)
- **Язык:** английский
- **Тон:** честный, data-driven, practical
- **Crave:**
  - Solo-мастера: витрина вместо Instagram DM-хаоса, автоматические напоминания, онлайн-оплата
  - Салоны: больше клиентов, CRM, аналитика, управление командой
- **CTA:** register provider → demo call → early-bird tariff (Free tier free forever)
- **Формат:** длинные text posts, инфографика, YouTube tutorials, AI-dev контент для developers

### 🧑‍💼 B2C — клиенты (RU primary)

- **Платформы:** Instagram Reels, TikTok, Facebook
- **Язык:** русский
- **Тон:** лёгкий, визуальный, «своя» девушка/парень
- **Crave:** найти мастера на своём языке · быстро забронировать · видеть все записи в одном месте · доверие через отзывы
- **CTA:** install app → первое бронирование
- **Формат:** Reels 30-60с, carousels, Stories, TikTok story-скетчи

## Пропорция форматов — НЕ только видео

Половина контента — текст и статическая графика. Это ускоряет производство и снижает зависимость от съёмочных дней.

| Формат | Доля | Постов (40+ за 8 нед) | Платформы |
|--------|------|---------------------|-----------|
| **Текстовые посты** (long-form) | 30% | 12 | LinkedIn, Facebook, Twitter |
| **Short-form video** | 30% | 12 | TikTok, IG Reels, YT Shorts |
| **Carousels / infographics** | 20% | 8 | Instagram, LinkedIn, Facebook |
| **Long-form video** | 10% | 4 | YouTube |
| **Stories / ephemeral** | 10% | 4 | Instagram Stories (ежедневно) |

## KPI tracking

Еженедельный отчёт в `docs/calendar/weekly-reports/week-NN.md` с метриками по каналам — формат в `docs/analytics/metrics-feedback-loop.md`.

Основные метрики:
- **Reach** (охват уникальных)
- **Engagement rate** (likes+comments+shares / views)
- **Saves** (сигнал качественного контента)
- **Link clicks → install / provider signup**
- **Cost per install / Cost per provider** (когда запустим Ads)
- **Community response rate** — 95% / <4ч в рабочее время
