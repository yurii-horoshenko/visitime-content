# Content Repurposing — один хит в 5 форматов

**Правило:** если пост получил ≥ 2× среднего ER за первые 48 часов, он автоматически кандидат на репурпосинг.

**Почему:** один хороший пост-хит работает на новую аудиторию когда пересобран в другом формате. Экономия 70% времени vs создание с нуля. Instagram-юзер не пересекается с LinkedIn-юзером в 80% случаев.

---

## Матрица репурпосинга

Для каждого исходного формата — 4 целевых.

### Исходник: LinkedIn long-form EN (текст)

| Целевой формат | Как делаем |
|---------------|------------|
| **Twitter thread** | Разбиваем текст на 5–8 твитов, усиливаем крючок |
| **Instagram carousel** [RU] | Переводим 5–7 ключевых поинтов на RU, делаем слайды |
| **TikTok talking head** [EN/RU] | Ты озвучиваешь пост на камеру 60с, берём 3 главных поинта |
| **YouTube Short** | Длинная версия TikTok видео, 60с, с субтитрами на 2 языках |

### Исходник: TikTok скетч RU

| Целевой формат | Как делаем |
|---------------|------------|
| **Instagram Reel** | Перепост с изменённой обложкой и CTA |
| **YouTube Short** | Перепост + английские субтитры |
| **Twitter video** | Встраиваем в тред про проблему |
| **Facebook post** | Скриншот ключевого кадра + текстовое описание |

### Исходник: Instagram carousel RU

| Целевой формат | Как делаем |
|---------------|------------|
| **LinkedIn carousel EN** | Переводим + адаптируем под B2B-аудиторию |
| **TikTok** | Snapshot-видео — каждый слайд 2–3 сек, музыка, text-on-screen |
| **Facebook post** | Лонгрид с изображениями |
| **Twitter thread** | Один слайд = один твит |

### Исходник: YouTube long-form tutorial EN

| Целевой формат | Как делаем |
|---------------|------------|
| **YouTube Shorts** (3–5 штук) | Нарезаем лучшие моменты по 60с |
| **TikTok** (3 штуки) | Те же нарезки RU-переводом |
| **LinkedIn post** | Текстовый summary + ссылка на полное видео |
| **Instagram carousel** | Ключевые этапы tutorial как слайды |

---

## Правила репурпосинга

1. **Не раньше 48ч** после оригинала — даём посту отработать
2. **Не одновременно** с репурпосом на одной платформе — разнос минимум 3 дня
3. **Адаптируем крючок** под платформу — один и тот же пост на TikTok и LinkedIn начинается по-разному
4. **Меняем визуал** — не копируем один в один обложку, иначе алгоритм видит дубль
5. **Трекаем отдельно** — у каждого репурпосинга свой UTM, чтобы понимать эффективность

## Как запустить

Автоматически (рекомендуется):
- Триггер из `docs/analytics/metrics-feedback-loop.md`
- Агент `content-creator` получает команду repurpose автоматически

Вручную (если нужно срочно):

```
репурпосни пост #N в 4 формата — IG carousel RU, Twitter thread, TikTok 30с, LinkedIn carousel EN
```

---

## Архив hit-ов

Все посты-хиты (ER ≥ 2× avg) складываются в `content/hits/` — это референс-библиотека что работает в нашей аудитории. Пересматриваем раз в месяц и учимся.
