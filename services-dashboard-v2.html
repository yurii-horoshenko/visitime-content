🔑 Стратегия доступов — без shared-паролей
Идея: отдельный marketing.visitime@gmail.com не заводим — это создаёт третий ящик без ясной зоны ответственности. Доступ даётся через роли и SMM-tool, а не через общий пароль. marketing@visitime.net (Cloudflare-алиас) закрывает все нужды: bio-контакт, логин в SMM-tool, уведомления от рекламных кабинетов.

3 паттерна доступа:
· Platform-native delegation — основной способ для Meta BM, Google Ads, GA4, TikTok Business Center, YouTube, GBP, LinkedIn. Маркетолог заходит под своим email, пароли *.helpdesk никто не трогает.
· Shared login через Bitwarden Collection — для Instagram/TikTok (нативные приложения для съёмки Reels и контента), пароль не виден текстом.
· SMM-tool с OAuth — один инструмент (Buffer $15 / Metricool €18 / Publer) подключается ко всем соцсетям, маркетолог и AI-агенты работают только там.
🤖 Автопостинг — статус каналов
Реальность: AI-агент не постит сам напрямую — он генерирует контент и вызывает API платформ. Токены выдаются один раз, хранятся в секретах, оркестрирует n8n / Make.com / cron. Ниже — по каким каналам автопостинг работает «из коробки», а где нужны костыли.
Сервис Статус API / что можно Что нужно для доступа
📘 Facebook Page✅ ReadyGraph API · posts (text/photo/video/link), scheduled, carouselsPage Access Token (long-lived → вечный) через Meta BM
📸 Instagram Business✅ ReadyGraph API · feed photo/video, Reels, carouselsBusiness account + связка с FB Page + тот же Meta token

📍 Google Business Profile✅ ReadyBusiness Profile API · posts, events, offers, обновленияOAuth Google + верифицированный профиль
💼 LinkedIn Company Page✅ ReadyMarketing/Share API · text + media posts от имени компанииPartner access (авто для Company admin) + OAuth
🎵 TikTok Business⚠️ ApprovalContent Posting API · видео, photo carousels, draftsЗаявка approved app, review 2–4 недели. До аппрува — Metricool
💬 WhatsApp Business⚠️ TemplatesCloud API (Meta) · outbound только approved шаблоны; входящие автоответы свободноMeta WhatsApp Business Platform + approval шаблонов
📢 Meta Ads / 🎯 Google Ads / 📢 TikTok Ads➖ N/AAPI для управления рекламными кампаниями, не органикойОтдельная автоматизация — не постинг
📨 Cloudflare · 📈 GA4 · 🎯 Meta Pixel➖ N/AИнфраструктура / аналитика — не контентные каналы—
Рекомендуемый сетап автопостинга:
· Оркестратор — n8n (self-hosted, free) или Make.com ($9+/мес) или cron + Python/Node
· Секреты — META_PAGE_ACCESS_TOKEN, GOOGLE_OAUTH_REFRESH_TOKEN, LINKEDIN_ACCESS_TOKEN, TIKTOK_ACCESS_TOKEN в Bitwarden / .env
· AI-агент — генерация постов, caption + hashtags, ресайз под форматы (1:1 / 4:5 / 9:16), best-time по TZ Warsaw, A/B варианты, анализ метрик
· SMM-tool fallback — Metricool €18/мес закрывает TikTok без approval и все 5 ready-каналов через OAuth (если не хочешь возиться с API напрямую)
Поэтапный план запуска автопостинга:
Неделя 1 — Meta (FB + IG) + Google (YouTube + GBP) через прямые API → 4 канала, 80% трафика  ·  Неделя 2 — LinkedIn (B2B)  ·  Параллельно — заявка на TikTok Content Posting API, до аппрува через Metricool  ·  Когда будет база номеров — WhatsApp template-бродкасты через Meta Cloud API
