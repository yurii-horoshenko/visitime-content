💡 Как это работает на практике
Ты в Claude Code пишешь: подготовь контент на неделю 3. Перед этим (Фаза 00) я уже собрал метрики прошлой недели и обновил план. Далее product-manager → задачи в Linear → агенты пишут тексты, делают карусели и сценарии с опорой на brand voice guide. Ты подключаешься в фазах 04 (съёмка / апрув текстов) и 06 (final review). После публикации — Фаза 08 community management 15+15 мин/день на ответы. Hit-посты автоматически репурпосятся и триггерят paid boost.

Pipeline — 9 фаз (замкнутый цикл)

00
📊 Анализ прошлой недели NEW
Feedback loop — метрики прошлой недели определяют план следующей
Агент делает сам
Каждый понедельник в 08:00 analytics-specialist пулит метрики по всем каналам, ранжирует посты (🔥 hit / ✅ solid / 😐 avg / ❌ flop), даёт 3 рекомендации на следующую неделю. Hit-посты автоматически идут в Repurposing и триггерят Paid boost уведомление.
Вход: metrics API  →  Выход: weekly report + 3 adjustment для product-manager
Агенты analytics-specialist seo-specialist
От тебя Ничего — отчёт сам появится в docs/calendar/weekly-reports/. Читаешь 5 мин в понедельник.
01
📋 Планирование недели
Решаем что именно публикуем на конкретную неделю
Я запускаю
Говоришь: «подготовь контент на неделю N» или «запускаем план недели 3». Я открываю docs/calendar/NN-week-XX.md, подтверждаю темы и передаю product-manager для декомпозиции.
Вход: календарь недели  →  Выход: 7 задач VIS-XXX в Linear со статусом Todo
Агенты product-manager linear-task
От тебя Команда «подготовь контент на неделю N». Всё.
02
🎯 Стратегия и хуки
Уточняем под платформы, находим крючки, ключевые слова
Агент делает сам
social-media-strategist адаптирует темы под специфику каждой платформы. tiktok-strategist подбирает trending sounds и первые 3 секунды для скетчей. seo-specialist — хэштеги RU и EN.
Вход: темы из плана  →  Выход: уточнённые хуки + хэштеги + trending сценарии в задачах Linear
Агенты social-media-strategist tiktok-strategist seo-specialist
От тебя Ничего — фоновая фаза.
03
✍️ Генерация контента
Пишем тексты, сценарии, делаем макеты каруселей
Агент делает сам
Разделение по типу контента:
· Текстовые посты (LinkedIn, Twitter, Facebook) — content-creator пишет финальные тексты на EN/RU
· Карусели и инфографика — instagram-curator структура + visual-storyteller макет в Figma
· Видео-сценарии — narratologist 5-актовая структура → content-creator сценарий → short-video-editing-coach shotlist
Вход: уточнённые темы  →  Выход: файлы в content/posts/ · content/graphics/ · content/videos/scripts/
Агенты (по типу) content-creator narratologist instagram-curator visual-storyteller short-video-editing-coach
От тебя Ничего — агенты пишут сами. В конце фазы увидишь готовые файлы.
04
🎬 Твоё участие — съёмка и апрув
Единственная фаза где ты активно работаешь руками
Я делаю сам
Что от тебя требуется:
· Прочитать тексты постов 1–4 недели, где EN/RU тексты готовы — поправить если звучит «не по-твоему»
· Снять видео — по shotlist'у (обычно 1 съёмочный день в неделю). Скринкасты делаешь сам с телефона, без оператора
· Одобрить макеты каруселей в Figma одним «ок»
Вход: тексты, сценарии, shotlist  →  Выход: raw footage в production/raw-footage/ + одобренные тексты
Агенты в помощь video-optimization-specialist visual-storyteller
От тебя ~2–4 часа в неделю: чтение текстов, съёмочный день (4–5ч раз в 1–2 недели), апрув каруселей.
05
🎞️ Пост-продакшн видео
Монтаж, субтитры, адаптация под форматы 9:16, 1:1, 16:9
Агент делает сам
video-optimization-specialist берёт raw footage → делает монтаж (или даёт тебе precise cut-list для CapCut), добавляет burned-in субтитры RU/EN, делает обложки, ресайз под форматы платформ, выгружает в production/edited/.
Вход: raw footage  →  Выход: финальные видео 9:16 (TikTok/IG) · 1:1 (FB) · 16:9 (YouTube) + thumbnails
Агенты video-optimization-specialist short-video-editing-coach
От тебя Ничего, если монтаж делает агент. Если руками в CapCut — следуешь cut-list (~30–45 мин на видео).
06
✅ QA Review — финальная проверка
Чек-лист перед публикацией: смысл, орфография, бренд, CTA
Мой апрув
qa-engineer проверяет по Acceptance Criteria: правильный хук в первых 3 сек, корректные хэштеги, рабочая ссылка в bio, субтитры читабельны, нет опечаток, соответствие бренду. Даёт отчёт «ready / нужны правки». Ты утверждаешь одним сообщением — и пост уходит в очередь.
Вход: финальные материалы  →  Выход: статус задачи Linear → Ready to publish
Агенты qa-engineer qa-verify
От тебя Последний взгляд на готовый пост. Одна минута на пост.
07
🚀 Публикация по расписанию + auto-triggers
Автоматический постинг через API платформ в нужное время
Автоматика (API / cron)
Оркестратор (n8n или cron + скрипт) берёт одобренный пост и публикует через API. Перед публикацией проверка blackout-dates — если сегодня траур / кризис, пост автоматически переносится. Через 48 часов срабатывают auto-triggers: hit-пост (ER ≥ 2× avg) → notification «buster?» + repurposing candidates.
Вход: одобренный пост  →  Выход: публикация + ссылка в Linear + 48h auto-check на repurpose/boost
Инструменты n8n / Make.com Meta Graph API YouTube API LinkedIn API Metricool (TikTok)
От тебя Ничего. Единственное — один клик «yes, boost» если пост взлетел.
08
💬 Community engagement NEW
Отвечаем на комменты и DM пока горячо — 4ч в рабочее время, 24ч макс
Я + агент
Ежедневный ритуал 15+15 мин — утром и вечером. community-manager проверяет все каналы (IG DM, TikTok comments, LinkedIn, FB, Twitter), отвечает по saved replies, эскалирует тебе только сложные случаи. Лиды B2B попадают в Linear как VIS-LEAD-XXX, баги — к dev-команде.
Вход: комменты + DM после публикации  →  Выход: ответы в 4ч, лиды в Linear, метрики в weekly report
Агенты community-manager content-creator
От тебя ~10 мин/день на сложные ответы (публичная критика, лиды, legal, баг-репорты). Рутину закрывает агент.
